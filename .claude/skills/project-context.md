# 琳達髮廊預約系統 - 專案背景資訊

## 基本資訊

### 店家名稱
- **中文名稱**: 琳達髮廊
- **英文名稱**: Linda Hair Salon

### 專案目標
- 建立一個手機端優化的線上預約系統
- 部署在 AWS Amplify
- 目前階段使用假資料，不串接 API

## 技術規格

### 技術棧
- **前端框架**: Next.js 14
- **樣式**: Tailwind CSS
- **部署平台**: AWS Amplify
- **目標裝置**: 主要為手機端

### 設計原則
- Mobile-first 設計
- 響應式介面
- 簡潔易用的預約流程
- 清晰的視覺層級

## 功能需求

### 已實作功能
1. **服務展示頁** (首頁)
   - 8 種美容服務展示
   - 服務分類篩選
   - 服務詳情顯示

2. **預約流程** (4 步驟)
   - 步驟 1: 選擇服務
   - 步驟 2: 選擇設計師
   - 步驟 3: 選擇日期與時間
   - 步驟 4: 填寫聯絡資訊

3. **預約確認頁**
   - 顯示完整預約詳情
   - 預約成功確認

### 服務項目
- **重要**: 服務項目將從後台 API 動態取得
- 目前使用假資料作為展示（data/services.js）
- 未來需要串接後台商品管理 API
